# Ли Бо — Чайный Бар

Веб-приложение для чайного бара «Ли Бо». Представляет собой интерактивное меню с возможностью подбора напитка по вкусовым предпочтениям и работает как Progressive Web App.

---

## О проекте

«Ли Бо» — это чайный бар, вдохновлённый китайской культурой. Приложение позволяет гостям изучить меню, получить персональную рекомендацию напитка, а также узнать информацию о баре — часы работы и контакты.

Приложение написано на чистом HTML, CSS и JavaScript без сторонних фреймворков. Оно поддерживает офлайн-режим через Service Worker и может быть установлено на мобильное устройство как обычное приложение.

---

## Функциональность

**Меню**

Четыре категории позиций с фотографиями и полными описаниями:

- Холодные коктейли
- Горячие коктейли
- Алкогольные коктейли
- Блюда

Каждую позицию можно открыть в детальном просмотре с галереей изображений, описанием и ценой.

**Подбор напитка**

Пять вопросов о вкусовых предпочтениях — вкус, температура, наличие алкоголя, настроение, вкусовые ноты. По результатам алгоритм подбирает три наиболее подходящих напитка.

**Информация о баре**

Режим работы, номер телефона и краткое описание заведения.

**Офлайн-режим и установка**

Service Worker кеширует ресурсы приложения. При наличии интернет-соединения данные обновляются автоматически; при его отсутствии приложение продолжает работать из кеша. Поддерживается установка на рабочий стол или главный экран через браузер.

## Технологии

| Технология         | Применение                                   |
|--------------------|----------------------------------------------|
| HTML5              | Разметка                                     |
| CSS3               | Стилизация, адаптивная вёрстка               |
| JavaScript         | Вся логика приложения                        |

## Локальный запуск

Сборка не требуется. Достаточно запустить любой локальный HTTP-сервер в корне репозитория:

```bash
# Python 3
python -m http.server 8000

# Node.js
npx http-server

# PHP
php -S localhost:8000
```

После запуска открыть в браузере: `http://localhost:8000`

> Открытие `index.html` напрямую через `file://` тоже работает, однако Service Worker в этом режиме не регистрируется.

---

## API 

По умолчанию приложение использует данные из локальных файлов `menu-data.js` и `quiz-data.js`. При необходимости можно подключить внешний API, указав его базовый URL в `js/app.js`:

```javascript
const API_BASE = 'https://admin.libo.bar/api';
```

Ожидаемые эндпоинты: `GET /api/menu` и `GET /api/quiz`. При недоступности API приложение автоматически откатывается к локальным данным.

## Дизайн

Тёмная цветовая схема с акцентами на читаемость. Типографика основана на шрифтах с поддержкой китайских символов (Noto Serif SC, Noto Sans SC). Вёрстка адаптирована для мобильных устройств и учитывает safe-area у устройств с вырезом экрана.
